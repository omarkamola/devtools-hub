# Security Policy

## Reporting a Vulnerability

If you discover a security issue in DevToolsHub, please report it responsibly.

Please do not open public GitHub issues for security vulnerabilities.

Instead, contact the project maintainer with:
- a description of the issue
- steps to reproduce
- potential impact

Most tools in DevToolsHub run entirely in the browser and do not send user data to external servers.

Reports will be reviewed and addressed as quickly as possible.

Thank you for helping improve the security of DevToolsHub.
